Prérequis
---------

Installation
------------
Depuis une console (PowerShell sur Windows) :

```powershell

python -m venv .venv
.\.venv\Scripts\Activate.ps1
# (si des dépendances sont nécessaires, installez-les ici)
```

Exécution
---------
Pour lancer le programme :

```powershell
python main.py
```
